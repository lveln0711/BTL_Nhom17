## Cách chạy chương trình

### Yêu cầu
* JDK 17 trở lên
* JavaFX SDK
* IDE hỗ trợ JavaFX (Eclipse / IntelliJ)

### Chạy chương trình

Chạy file:

```text id="7emzxm"
src/ui/MainApp.java
```

Nếu cần cấu hình JavaFX VM Arguments:

```text id="jlwmw5"
--module-path "PATH_TO_FX/lib" --add-modules javafx.controls,javafx.fxml
```
